#ML #Math 
ML systems uses **Relationships** between **Inputs** to produce **Predictions**. 
Relationship is written as **$y = b + wx$**:
	- **$y$** is the label we want to predict
	- **$w$** is the weight (the slope)
	- **$x$** are the features (input values)
	- **$b$** is the intercept
$y = b + w_1x_1 + w_2x_2 + w_3x_3 + w_4x_4$
**Model** defines the relationship between the **label** $y$ & the **features** $x$. Phases in the life of a model: Data Collection, Training, Inference.
**Inference** is when the trained model is used to infer (predict) values using live data. Like putting the model into production.
