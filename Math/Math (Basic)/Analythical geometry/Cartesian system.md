#Math 
**Cartesian product:** **$A \times B$** = set of **ordered pairs** $(a, b)$, where $a \in A$, $b \in B$  
$A = {1, 2}$, $B = {a, b, c}$ $A \times B = {(1, a), (1, b), (1, c), (2, a), (2, b), (2, c)}$
**In Cartesian orthogonal [[Coordinate system]]:** $\vec{a} \cdot \vec{b} = a_x b_x + a_y b_y (+ a_z b_z)$  
$|\vec{a}| = \sqrt{a_x^2 + a_y^2 + a_z^2}$
