$f'(c) = \lim\limits_{x \to c} \dfrac{f(x) - f(c)}{x - c}$
**Symmetric derivative:** $\lim\limits_{h \to 0} \dfrac{f(x + h) - f(x - h)}{2h}$
**Monotonicity**:
- If $f'(x) > 0 \Rightarrow f(x)$ is **increasing**
- If $f'(x) < 0 \Rightarrow f(x)$ is **decreasing**

$(\ln x)' = \dfrac{1}{x}$
$(e^x)' = e^x$
$(x^n)' = nx^{n - 1}$
$(a^x)' = a^x \ln a$
$\left(\sin^{-1} x\right)' = \dfrac{1}{\sqrt{1 - x^2}}$, $|x| < 1$
$(\cos x)' = -\sin x$
$(\tan x)' = \dfrac{1}{\cos^2 x}$
$(\cot x)' = -\dfrac{1}{\sin^2 x}$
$(\log_a x)' = \dfrac{1}{x \ln a}$
$(ku)' = k u'$ 
$(k)' = 0$
$(uv)' = u'v + uv'$
$(u + v)' = u' + v'$
$\left(\dfrac{u}{v}\right)' = \dfrac{u'v - uv'}{v^2}$
$(\sqrt{u})' = \dfrac{u'}{2\sqrt{u}}$
