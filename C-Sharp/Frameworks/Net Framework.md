#CSharp #SoftDev 
**.NET Framework** is a softdev Platform dev by Microsoft for building & running Windows apps. It provides a **large class library** & supports several programming langs, primarily **C#**, **VB.NET**, & **F#**. Windows-only.

**CLR** – Common lang Runtime
The exec engine for .NET apps.
- Manages memory, threads, & exception handling
- Provides **garbage collection**, **type safety**, **JIT compilation**    
- Exec **MSIL** (Microsoft Intermediate lang) code    
**FCL** – Framework Class Library, a collection of reusable classes, interfaces, & value types.
- Namespaces: `System.*`, `Microsoft.*`    
- Includes support for:    
    - File I/O (`System.IO`)        
    - Networking (`System.Net`)        
    - Security (`System.Security`)        
    - Collections (`System.Collections`)        
    - XML (`System.`XML)       
    - Windows Forms, WPF, ASP.NET
**Key Tech Built on .NET Framework**
- **ASP.NET** – Web apps & services    
- **WinForms** – Desktop GUI apps    
- **WPF** – Advanced desktop UI (XAML-based)
- **ADO.NET** – Data access & DB communication    
- **[[Entity framework]]** – ORM for data modeling
