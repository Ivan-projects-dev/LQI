**Multithreading** in C# is a tech that allows a program to perform multiple operations at the same time by exec different threads concurrently. It is commonly used to improve performance, responsiveness, & efficient resource utilization in apps.

C# provides multithreading support through the `System`. Threading namespace & higher-level abstractions like **[[TPL]] ([[TPL]])** & [[Async & await]].
